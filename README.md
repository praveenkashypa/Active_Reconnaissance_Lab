# Active_Reconnaissance_Lab
**Overview**
This repository contains a PDF walkthrough of the "Active Reconnaissance" room on TryHackMe, a hands-on cybersecurity training platform. The PDF is a 6-page compilation of screenshots capturing the room's interface, tasks, commands executed in the AttackBox terminal, questions, and correct answers submitted. It serves as proof of completion and a visual guide for learners exploring active reconnaissance techniques, which involve direct interaction with targets (e.g., sending packets) to gather information.
**Purpose**
This PDF is useful for:

Reviewing active recon fundamentals, such as network probing without exploitation. Serving as study notes for cybersecurity beginners Demonstrating progress in TryHackMe's learning paths, particularly in the Pre Security or Introduction to Offensive Security modules.
